# ShopVista E-commerce

![ShopVista](https://picsum.photos/id/1015/1200/400)

A beautiful, fully functional modern e-commerce website built with **HTML5**, **Tailwind CSS** (via CDN), and **vanilla JavaScript**. 

**Live Demo:** Simply open `index.html` in any modern browser (or enable GitHub Pages for hosted version).

## ✨ Features

- **Elegant & Responsive Design** — Mobile-first, premium clean UI with smooth interactions and micro-animations
- **Product Catalog** — 12 curated premium products across 6 categories with high-quality imagery
- **Advanced Filtering & Search** — Real-time search, category filters, sorting by price, rating, name
- **Quick View Modal** — Detailed product view with dynamic size/color selection and quantity
- **Shopping Cart Drawer** — Persistent cart (localStorage) with live quantity adjustment and totals
- **Seamless Checkout Experience** — Beautiful checkout modal with form validation simulation and success confirmation
- **User Session Simulation** — Login/Register modal with persistent user state in navbar
- **Wishlist Functionality** — Toggle favorites with dedicated modal
- **Toast Notifications** — Elegant feedback messages for add to cart, etc.
- **Fully Client-Side** — No server or build step required. Works great offline after initial load.

## 🛠 Tech Stack

- Tailwind CSS (Playground CDN)
- Pure Vanilla JavaScript (ES6+)
- Font Awesome 6 for icons
- Browser LocalStorage for cart & authentication persistence

## 🚀 Getting Started

### Local Development
1. Clone the repository: `git clone https://github.com/dasucosmos-eng/shopvista-ecommerce.git`
2. Open the folder and double-click `index.html` (or right-click → Open with Live Server in VS Code)
3. Enjoy the fully interactive experience!

### Deploy to GitHub Pages (Free Hosting)
1. Go to repository **Settings** → **Pages**
2. Under "Build and deployment", select **Source**: Deploy from a branch
3. Choose branch: `main` and folder: `/ (root)`
4. Save — your site will be live at `https://dasucosmos-eng.github.io/shopvista-ecommerce`

## 📝 How to Customize

- Edit the `products` JavaScript array in `index.html` to add/remove products
- Change brand name, colors (via Tailwind config in script), or add real payment integration
- Extend with backend using Stripe, Supabase, or Shopify Hydrogen for production
